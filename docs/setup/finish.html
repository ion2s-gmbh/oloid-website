# Finishing touches

Finally, you finish the installation by running:

php artisan workshop:install

This command will configure your Laravel project to use oloid-laravel-patternlib and publish some of our assets/resources.

Update the package

Whenever you update the package a newer version, you should rerun php artisan workshop:install. This will publish the new package's assets.
